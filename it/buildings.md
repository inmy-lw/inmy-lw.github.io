# Gestione delle caserme

<a href="https://youtu.be/ajbNCrJF5N0?t=547" target="_blank">Video esplicativo</a>

Il livello delle caserme deve essere “scalare”: ogni caserma deve poter produrre un livello di unità diverso.

Se puoi produrre unità di livello 8, allora devi avere:
1. Una caserma di livello 24-26 (che può formare T8)
2. Una caserma di livello 20-23 (che può formare T7)
3. Una caserma di livello 17-19 (che può formare T6)
4. Una caserma di livello 14-16 (che può addestrare T5)

Il principio è quello di addestrare per ogni caserma unità di livello inferiore di 1 rispetto alla caserma. Nel caso precedente, addestreremo:
1. T7
2. T6
3. T5
4. T4

E questo per tutta la settimana, fino al venerdì, quando le caserme serviranno solo per le promozioni:
1. Da T7 a T8
2. T6 verso T7
3. T5 verso T6
4. T4 verso T5
