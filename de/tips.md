# Ungewolltes Öffnen von Geschenken vermeiden

Das Öffnen von Geschenken (Bestätigung des Bauabschlusses eines Gebäudes) bringt viele Punkte für den VS. Daher solltest du möglichst viele Geschenke für den Dienstag aufsparen. Es gibt eine Einstellung, die das versehentliche Öffnen verhindert:

{% include image.html url="/assets/img/tips/confirmation_ouverture_cadeau.webp" description="Bestätigung beim Öffnen eines Geschenks erhalten" %}

Sobald diese Einstellung aktiv ist, erscheint dieses Bestätigungs‑Popup, wenn du ein Geschenk berührst

{% include image.html url="/assets/img/tips/popup_confirmation_cadeau.webp" description="Bestätigungs‑Popup" %}
