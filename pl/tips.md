# Unikać przypadkowego otwierania prezentów

Otwieranie prezentów (akceptacja zakończenia budowy) daje dużo punktów do VS. Dlatego najlepiej mieć jak najwięcej prezentów do otwarcia we wtorek. Istnieje ustawienie zapobiegające przypadkowemu otwarciu prezentów:

{% include image.html url="/assets/img/tips/confirmation_ouverture_cadeau.webp" description="Potwierdzenie przy otwieraniu prezentu" %}

Po włączeniu tego ustawienia ten popup potwierdzenia pojawi się, gdy dotkniesz prezentu

{% include image.html url="/assets/img/tips/popup_confirmation_cadeau.webp" description="Okno potwierdzenia" %}
